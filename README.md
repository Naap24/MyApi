# MyApi - Simple User Management API

This is a simple ASP.NET Core Web API project demonstrating CRUD operations
(Create, Read, Update, Delete) for managing users.

## 🚀 Features
- GET /api/users → Retrieve all users
- GET /api/users/{id} → Retrieve a single user
- POST /api/users → Create a new user
- PUT /api/users/{id} → Update an existing user
- DELETE /api/users/{id} → Delete a user

## 🛠 Tech Stack
- .NET 8
- ASP.NET Core Web API
- In-memory repository (replaceable with EF Core)
